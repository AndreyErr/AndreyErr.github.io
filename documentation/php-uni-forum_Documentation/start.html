Необходимое ПО

Для простого запуска проекта вам потребуется программа Docker! Скачать ёё можно здесь для windows или следовать следующим инструкциям для установки на Linux. Программа сама подгрузит всё необходимое! Если возникнут ошибки, то обратитесь к документации докера или иным форумам.

Настоятельно рекомендуем научится пользоваться докером.

Желательна установка Git.

Дальнейшие действия будут описаны в следующей главе.

Начальные шаги

После подготовки необходимого ПО следуем следующим простым шагам:

1

Скачиваем репозиторий

Git

Выполните команду 

git clone https://github.com/AndreyErr/php-uni-forum.git
из нужной вам дерриктории.

Вручную

Скачайте репозиторий из последнего релиза или самую новую версию (может быть нестабильной) и разархивируйте в нужную вам деррикторию.

Установка контейнеров

Запускаем докер.

После запуска, в корневой дерриктории проекта php-uni-forum/ выполняем команду 

docker compose build
. После этого начнётся сборка проекта.

2

Запуск проекта

Запуск докер контейнеров

Для запуска контейнеров в корневой дерриктории проекта php-uni-forum/ выполняем команду 

docker compose up
.

Дождитесь завершения запуска (обычно не более 20 сукунд).

Остановить контейнеры можно в терминале сочетанием клавишь0 Ctrl + c!

Открыть проект

В проекте существуют следующие порты:

Заданные порты (по умолчанию)

Порт Описание Ссылка
8080 Основной порт через который происходит управление проектом. На этом порту находится nginx. http://localhost:8080
8082 Порт с визуализацией бд при помощи phpMyAdmin. http://localhost:8082
8025 Тестовый почтовый сервер (не является полноценным почтовым сервером). Но нём можно принимать отправленную с сайта почтую http://localhost:8025
8081 Apache сервер. Работает через nginx. Недоступно (НЕ ЗАНИМАТЬ)
1025 Порт для работы почтового сервера. Недоступно (НЕ ЗАНИМАТЬ)
1434 Порт для работы баз данных. Недоступно (НЕ ЗАНИМАТЬ)

Данные для входа (по умолчанию)

Место Ссылка Данные
Аккаунт админа http://localhost:8080 Admin, qwerty123#
Аккаунт на визуализаторе баз данных http://localhost:8082 Mysql_db, root, root
